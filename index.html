標準コースタイム 6時間48分（休憩含まず）。休憩込みの総行動時間は 約7時間30分〜8時間 を想定。

  • 後半（清滝→ゴール）: 距離 7.0km / 目安 3時間20分（標高差大）

  • 日没時刻を確認し、遅くとも日没1時間前にはゴールすること

危険箇所・注意事項
  • 松尾山手前の急登: 序盤から急勾配。ウォームアップ不足での心拍急上昇に注意。
  • 嵐山市街の車道横断: 観光バス・タクシー往来あり。横断歩道を利用すること。
  • 清滝トンネル: 車両通行あり。歩行者用通路が狭い。ヘッドランプ点灯推奨。
  • 清滝→西山峠の急登: コース核心部。岩場・鎖場あり。雨後は極めて滑りやすい。
  • 西山峠付近のやせ尾根: 両側が切れ落ちた箇所あり。強風時は特に注意。
  • 下山路の九十九折: 落ち葉で滑りやすい。膝への衝撃に注意。ポール使用推奨。
野生動物・害虫
  • ニホンザル: 嵐山周辺に生息。食料を見せない・目を合わせない。
  • イノシシ: 西山山中に出没例あり。遭遇時は静かに後退。
  • マダニ: 春〜秋に活動。長袖長ズボン着用。下山後の全身チェック必須。
  • スズメバチ: 夏〜秋。黒い服・香水を避ける。巣に近づかない。
  • ヤマビル: 雨後の低山で出没。塩・虫除けスプレーが有効。
コースマーキング

京都一周トレイルは京都市が整備した公式コース。標識は 赤地に白文字の番号標識（西山1〜西山XX）が 設置されている。分岐点には方向を示す矢印標識あり。

  • 標識間隔は概ね200〜500m。5分以上標識が見つからない場合はルートミスの可能性。
  • GPS（本アプリ）と紙地図を併用し、定期的に現在地を確認すること。
  • 不明瞭な分岐では必ず立ち止まり、ルートを確認してから進むこと。

消費カロリーの計算方法

本アプリでは以下の計算式（コース定数法）を使用しています。

C = 1.8×T + 0.3×D + 10.0×A_up + 0.6×A_down
消費カロリー = C × 体重(装備込)

  • T: 部分標準コースタイム（時間）
  • D: 歩行距離（km）
  • A_up: 累積上昇量（km）
  • A_down: 累積下降量（km）

救助要請の手順
  1. 安全な場所に移動（崖際・落石危険区域から離れる）
  2. 119番に電話。伝えること:
    • 「山岳事故です」
    • 場所: 「京都一周トレイル西山コース、西山○○番標識付近」
    • 本アプリの座標表示を読み上げ
    • 傷病者の状態・人数
    • 連絡可能な電話番号
  3. ホイッスルを6回吹く（国際山岳遭難信号）→1分待ち→繰り返し
  4. エマージェンシーシートで保温。動かさない（骨折疑い時）
  5. 電池を節約。機内モードにしてGPSのみONにする手もある。
